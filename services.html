Tor Exit Node Hosting

Without our servers, roughly 25% of all exit traffic in the Tor network passes through one node, which is far from ideal. The currently fastest node, Blutmagie, will be shut down within the next months, and the whole network will suffer heavily. We run a high bandwidth Tor exit node - one of the fastest nodes in existence - and several smaller nodes. This helps to make the Tor network more stable, faster and more anonymous.

Private Bridge IP addresses

Bridges can be used to access the Tor network if public entry nodes are blocked. The Tor website offers a limited number of bridge addresses, but they might get blocked fast (which is happening in China, for example), or monitored to track down its users.

We can provide exclusive private bridge IP addresses to individuals or organizations at prices starting from $1 per month and IP. You can also donate so we can provide free bridge IPs to those who need them. We work together with Torproject and other organizations to get them to the people. If you are an organization able to distribute bridge IPs locally, contact us!
